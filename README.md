setup
=====

Deployment Scripts for CentOS


## Installation

Execute the following commands at your servers command-line:

	$ yum -y install git
	$ git clone https://github.com/darcyliu/setup.git
	$ cd setup
	
###PPTP

	$ sh centos_pptp_install.sh

###LAMP(Apache)

	$ sh centos_lamp.sh
	
###LEMP(nginx)

	$ sh centos_lemp.sh
